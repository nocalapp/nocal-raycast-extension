# nocal Raycast Extension

Raycast companion extension for nocal.

## Status

This repository is being prepared as a store-ready Raycast extension. The initial scaffold is in place and the first commands are being implemented.

## Development

```bash
npm install
npm run dev
```

## Commands

- `Search Notes`
- `Create Note`
- `Upcoming Meetings`
- `Open nocal`

These are scaffolded entry points for the first release and will be fleshed out in follow-up specs.
